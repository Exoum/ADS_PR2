# Практическая работа №2: Поиск подстрок в строке

## Описание

Реализация различных алгоритмов поиска подстрок в строке с консольной утилитой для практического использования.

## Реализованные алгоритмы

1. **Кнута-Морриса-Пратта (KMP)** - основной алгоритм для поиска одной подстроки
2. **Бойера-Мура** - альтернативный алгоритм поиска
3. **Ахо-Корасик** - для поиска множества подстрок одновременно
4. **Рабина-Карпа** - с самостоятельно реализованным кольцевым хешем
5. **Бойера-Мура-Хорспула** - упрощенная версия алгоритма Бойера-Мура

## Возможности

- Поиск одной или нескольких подстрок
- Чувствительность/нечувствительность к регистру
- Поиск с начала или конца строки
- Ограничение количества результатов
- Поиск в файлах
- Цветовое выделение найденных подстрок
- Декоратор для измерения времени выполнения

## Использование

### Модуль search.py

```python
import search

# Поиск одной подстроки
result = search.search("ababbababa", "aba")
print(result)  # (0, 5, 7)

# Поиск нескольких подстрок
result = search.search("ababbababa", ["aba", "bba"])
print(result)  # {'aba': (0, 5, 7), 'bba': (3,)}
```

### Консольная утилита

```bash
# Поиск в строке
python main.py -s "ababbababa" aba

# Поиск в файле
python main.py -f input.txt aba bba

# Поиск с учетом регистра
python main.py -s "Hello World" -c hello

# Поиск с конца
python main.py -s "ababbababa" -m last aba

# Ограничение количества результатов
python main.py -s "ababbababa" -k 2 aba

# Отключение цветового выделения
python main.py -s "ababbababa" --no-highlight aba
```

## Параметры командной строки

- `-s, --string` - строка для поиска
- `-f, --file` - путь к файлу для поиска
- `-c, --case-sensitive` - учитывать регистр символов
- `-m, --method` - направление поиска (first/last)
- `-k, --count` - максимальное количество совпадений
- `--no-highlight` - отключить цветовое выделение
- `--max-lines` - максимальное количество строк для вывода
